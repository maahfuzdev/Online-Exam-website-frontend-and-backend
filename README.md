# Online Examination System

A full-stack web application for conducting online exams.

## Tech Stack
- **Frontend**: HTML/CSS/Bootstrap/Tailwind CSS/JavaScript 
- **Backend**: Node.js + Express
- **Database**: MongoDB 
- **Authentication**: Bcrypt

## Features
- User registration & login (Student + Teacher/Admin)
- Teachers can create exams with MCQ questions
- Students can attempt exams with timer
- Auto calculation of marks for MCQ
- View results
- View Previous Exam List and Result

## How to Run
1. Clone the repo
2. cd server && npm install
3. 
4. Create .env file in server folder
5. npx nodemon index.js

## Folder Structure
- /public → Frontend files
- /server → Backend (routes, models, controllers)
