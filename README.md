## 📌 Project Overview

A **comprehensive full-stack web application** designed to revolutionize the examination process by providing a seamless, secure, and efficient platform for conducting online exams. This system eliminates traditional paper-based examinations and offers real-time evaluation for both students and educators.

### ✨ Key Highlights
- 🔐 **Secure Authentication** - Role-based access (Student/Teacher/Admin)
- 📊 **Real-time Evaluation** - Automatic MCQ grading and instant results
- ⏱️ **Timed Examinations** - Built-in timer with auto-submission
- 📈 **Performance Analytics** - Track progress through previous results
- 🎨 **Modern UI** - Responsive design using Tailwind CSS & Bootstrap

## 📋 Features

### 👨‍🎓 For Students
- ✅ **User Registration & Login** - Secure account creation
- ✅ **Dashboard Overview** - View available exams
- ✅ **Attempt Exams** - Timed MCQ examinations
- ✅ **Real-time Timer** - Auto-submission when time expires
- ✅ **Instant Results** - Automatic score calculation
- ✅ **Previous Exams** - History of all attempted exams
- ✅ **Performance Tracking** - View scores and rankings

### 👨‍🏫 For Teachers/Admin
- ✅ **Secure Admin Login** - Role-based access control
- ✅ **Create Exams** - Add new exams with details
- ✅ **Add Questions** - Create MCQ questions with options
- ✅ **Set Timer** - Configure exam duration
- ✅ **View All Students** - Monitor student performance
- ✅ **Results Management** - View and export results
- ✅ **Exam Management** - Edit/Delete exams and questions

### 🔐 Security Features
- ✅ **Password Hashing** - Bcrypt encryption
- ✅ **Session Management** - JWT tokens
- ✅ **Role-based Authorization** - Protected routes
- ✅ **Input Validation** - Prevent injection attacks
- ✅ **XSS Protection** - Secure data sanitization
